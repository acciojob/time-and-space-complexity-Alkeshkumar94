Time,Space,Complexity

Time complexity measures how the execution time of an algorithm grows as the input size increases. Efficient algorithms strive to have lower time complexity, ensuring faster execution for larger inputs.

Space complexity refers to the amount of memory an algorithm uses as it runs. Algorithms with better space complexity consume less memory, which can be crucial in memory-constrained environments. One common notation to represent space complexity is O(n), where n is the size of the input.
